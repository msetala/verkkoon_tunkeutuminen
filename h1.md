# läksyjen palautus

### x) kooste

- Wiresharkia käytetään tietoliikenteen kaappaamiseen ja analysointiin
- Install komento:
```
sudo apt-get update
sudo apt-get install wireshark
```
- normaalikäyttäjän (ei superuserin) on annettava siepata paketteja, että ohjelma ajaa vähemmän koodia ylennetyillä oikeuksilla
- lisää käyttäjä wiresharkiin:
```
whoami
sudo user adduser meas wireshark
```
- avaa wireshark komennolla `wireshark`
<br></br>
- Network interface card ei välttämättä fyysinen verkkokortti: esim. loopback interface localhostille
- Interface prefixes:
-	**en** – Wired Ethernet
-	**wl** – WLAN
-	**lo** – Loopback adapter
- Loopback adapteri: yhteyksiin joita ei reititetä koneen ulkopuolelle. Jokainen kone kutsuu itseään localhost:iksi, IP 127.0.0.1 (IPv4 /8 | IPv6 ::1/128)
- interfacet näkee komennoilla:
```
ip a
ip route
```

- Frame 1: milloin kaappaus on aloitettu, mitä interfacea kaapataan (tässä tapauksessa enp0s3), encapsulation type (ethernet1)
- Ethernet II: **Link-Layer(Linkkikerros):** näyttää kohde MAC osoitteen sekä lähetyksen (source) MAC osoitteen. Fyysinen siirto
- IP-otsikko: **Verkkokerros.** Internet Protocol Version 4 -> source on oma tietokone ja destination 216.58.209.163 eli google. Paketointi ja reititys.
- **Kuljetuskerros**: Transmission Control Protocol: näyttää porttinumerot, source ja destination (80, HTTP – ei encrypted) Käynnistää ja lopettaa yhteyksiä.
- **Sovelluskerros** ei taida näkyä kuvakaappauksessa, mutta se olisi kohdat HTTP tai SSH.
